# Dolly's Closet - Product Catalog Website

## 📌 Overview
Client-based web application developed for a retail thrift store to improve its online presence through a structured product catalog and role-based system.

This repository represents the **customer-facing application**, built using modern web technologies.

- Inventory management workflows
- Role-based access system
- Customer-facing UI for shopping experience
